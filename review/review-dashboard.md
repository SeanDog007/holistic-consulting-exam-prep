## Review Workflow

### How to Review
1. Open a batch file from `review/batches/`
2. Work through each question — verify correct answer, check distractors, check for ambiguity
3. Mark each question: ✅ APPROVED, 🔧 REVISED, or 🚩 FLAGGED
4. After completing a batch, run `node review/update-tracker.mjs <batch-file>` to sync decisions back to tracker
5. Re-run `node review/generate-review-system.mjs` to refresh dashboard

### Recommended Review Order
1. Start with Domain 5 (smallest — 66 questions, 3 batches) to warm up
2. Then Domain 3 (112 questions, 3 batches)
3. Then Domain 2 (192 questions, 5 batches)
4. Then Domain 4 (175 questions, 3 batches)
5. Then Domain 1 (291 questions, 6 batches)
6. Finally: Full Length Comprehensive Exam (115 questions, 1 batch)
